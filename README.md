# Simply Weather

A clean, modern weather application built with Vanilla JavaScript, HTML, and Tailwind CSS. Simply Weather allows users to search for any city and get real-time, accurate weather data along with a 5-day forecast.

## Features

- **Current Weather Data**: Displays exact temperature, "feels like" temperature, daily high/lows, and weather conditions.
- **Dynamic 5-Day Forecast**: Automatically parses extensive API data to provide a clean, daily forecast snapshot.
- **In-Depth Metrics**: View real-time wind speed, wind gusts, wind direction, humidity, and local sunrise/sunset times.
- **Dynamic Icons**: Weather icons automatically update based on the current weather condition using Font Awesome.
- **Sleek UI**: Fully responsive and styled using Tailwind CSS for a modern, dark-mode aesthetic.

## Technologies Used

- **HTML5** \* **JavaScript (ES6+)**: Utilizes modern asynchronous JavaScript (`async/await`, `Promise.all`) for API fetching and DOM manipulation.
- **Tailwind CSS**: Used via CDN for rapid, utility-first styling.
- **Font Awesome**: For dynamic weather iconography.
- **OpenWeatherMap API**: Powers the real-time and forecast data.
